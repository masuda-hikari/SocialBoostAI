## 🔐 セキュリティ要件

### API認証管理
```
❌ ハードコード禁止
❌ ログ出力禁止
❌ Git commit禁止

✅ 環境変数(.env)で管理
✅ 最小権限の原則
✅ OAuth2フローでユーザートークン管理
✅ トークン暗号化保存
```

### ユーザーデータ保護
- マルチテナント設計（ユーザー間データ隔離）
- SQLite/PostgreSQLでユーザー別データ管理
- アクセストークンは暗号化保存

## ⚠️ 品質ガイドライン

### コード品質
- Type hints必須
- Docstring（Google style）
- black + isort でフォーマット
- mypy型チェック
- pytest カバレッジ80%+

### API利用
- レート制限厳守
- 指数バックオフ実装
- エラーハンドリング必須
- ログ出力（機密情報除外）

### コンテンツ生成
- プラットフォーム規約遵守
- スパム禁止
- ユーザー確認なしの自動投稿禁止
